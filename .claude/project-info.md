# Project Information

## プロジェクト目的
Discord BOTの開発

## 技術スタック
- 言語: JavaScript (Node.js)
- 主要ライブラリ: discord.js, dotenv
- 開発ツール: nodemon

## ブランチ構造
- `master` - 本番用ブランチ（安定版のみ）
- `develop` - 開発用ブランチ（機能開発のベース）
- `feature/*` - 機能開発用ブランチ（developから分岐）
